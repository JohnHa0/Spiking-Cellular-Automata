Conway's Game Of Life - User Manual





Button / Menu Operation
Start/stop Start or stop the running.
Speed Slider to determine the frame rate.
Grid size Determines the grid size. One can use the slider, or type manually.
Number of iterations Determine the number of iteration. One can use the buttons, or type manually.
While "infinite loop" were choosen, the other buttons are disabeled.
Style Determine the initial state of the grid.
For random, the user should determine the density.
For the circle or square, the user should input the size.
For the oscillators/gliders/spaceships, the user should choose if the grid will be single or multy samples.
Most of the gliders were taken from http://fano.ics.uci.edu/ca/rules/b3s23/.
Clear Clear the grid.
Inverse Turn the white cells into black, and vice versa.
Draw Let the user draw on the grid.
Reset Turn the grid to the state it was before running/drawing/clearing.
Move grid Let the user move the grid up/down/right/left, flip vertical or horizontal, ot transpose the grid.
Grid lines Show/hide the grid lines.
Load Loading MATLAB mat-file to the grid.
Save matrix Save the grid as MATLAB mat-file.
Save as image Save the grid as BMP, JPEG, PNG or TIFF image.
Save as video While start the running, the animation will be saved as an AVI video file.
The frame rate is determine by the initial speed.
Close Exit program.

The running stops in one of three cases:

  1. The user press the stop button.
  2. The program complete the number of iterations.
  3. The grid is in "steady-state".



Enjoy the Game!
